# Library Management System

## Objective
Develop a mini Library Management System using Java OOP concepts.

## Features
- Add Books
- Display Books
- Issue Books
- Return Books

## OOP Concepts Used
- Encapsulation
- Abstraction
- Inheritance (can be extended)
- Polymorphism

## Tools
- Java
- VS Code
- Terminal
